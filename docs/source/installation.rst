Установка
=========

Требования
----------

* Python 3.7+
* Pygame 2.0+

Установка
---------

1. Клонируйте репозиторий:

.. code-block:: bash

   git clone https://github.com/HumanIGG/pygame.git
   cd pygame

2. Установите зависимости:

.. code-block:: bash

   pip install pygame

3. Запустите игру:

.. code-block:: bash

   python main.py
